# StrikeTask

A simple **Todo List application** built using **HTML, JavaScript**, and **React + Redux**.
Add, delete, and mark tasks as completed with a clear interface.

## Features
- Add new tasks via input placeholder and "Add Task" button
- Delete tasks with the "Delete" button
- Mark tasks as completed using the "Done" button
- Completed tasks get a strike-through effect and the "Done" button is disabled
- State managed using **Redux** (for React version) or plain JS (for HTML version)

## Tech Stack
- React (JSX & Functional Components)  
- Redux & Redux Toolkit for state management  
- JavaScript 
- HTML & CSS / Tailwind CSS for styling  
- Vite  
- Fetch API / Axios for API calls
